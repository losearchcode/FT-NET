# FT-NET 局域网全链路纯净分享通讯域 —— 使用手册

## 📌 产品溯源及定位
FT-NET 是一个**高度极简、零安装、毫无广告与臃肿注册界面、即开即燃**的私有群组交互应用域。
无论是由电脑发往手机、手机发往电脑，还是各种跨操作系统的复杂设备群发任务，只要浏览器连在一起，大家即可随时构建出一个高度绝密的短效分享隔离层，并确保系统在解散之际自动爆破掉所有文件遗产。

---

## 🚀 极速上手体验指引

### 第 1 步：用“共同的意念暗号”开拓时空
- 抛开常规的“输入账号、加好友”，现在的启动大门异常冷酷干练：直接丢给您输入框。**这个密码也就是这间密室本身**！
  - **如果您是首达者**：随便输入 `000` 或异常变态长密码如 `Pla92$!`。一旦点解【加入】，该房间会在虚空服务器上以您的名义被实时凭空“生成创立”！第一人必定就是这间暗房的拓荒者。
  - **如何共享链接？** 直接喊出来、贴过去或是隔空扫码将这个密码口令发送给同网段（或您远端公网下）的朋友。只要他敲进去了一模一样的这段密码，就会在一瞬间凭空穿梭进您创立在这的隔离区。

### 第 2 步：多设备端对端云存储柜大区 (File Cabinet)
- 抵达房内的第一时间映入眼帘的便是置顶大板块——【服务器共享存储柜】。
- 不要局限于单传了，赶紧点中间或是把四五个需要互通图纸文件粗暴地往虚线框中心托拽上去。
- **神级提取法**：
  - 各个兄弟端不用去点收件了！云柜里面会瞬间列出所有全量留存档案。您可以单个点击右侧的那个隐晦的幽蓝色【⬇下载】按钮来拿。
  - 更加暴力的玩法：每个资料前有个隐秘的正方形勾选框。你可以开启“采购扫荡”直接点击顶部那个白净的【全选】按钮，然后按下【批量下载】一次打包拖空全线资料。假如是发错或机密合同，也同样可以用它点下【批量删除】，在秒间越级摧毁主服务器的原始资料数据区。
  - **快捷预览**：对于图片 (jpg/png/gif/webp/bmp/svg) 和 PDF 文件，每条记录右侧会多出一个紫色的【👁预览】按钮。点击后将弹出全屏毛玻璃遮罩预览窗口，图片以原尺寸自适应展示、PDF 以内嵌阅读器呈现，无需下载到本地即可快速查阅内容。点击遮罩层任意位置或右上角 ✕ 按钮即可关闭预览。

### 第 3 步：无限长弹幕化交流群录
- 处于全视图版面低端或被弹性滑窗牢牢束缚封印在那的，是一个能经受极快敲击和海量洪流承载的【文字密令厅】。
- 无论是三人进房进行三方通话，亦或八个人的乱入弹幕讨论，均辅以前瞻的排版行距约束与极具层次呼吸感的高端左右【色彩阵营气泡】渲染。自己和众人的聊天记录绝不混淆。而且滑动顺化永远不超出容器本身视觉底线。

### 🚨 第 4 步：阅后物理级“自毁连根拔除”禁术 (Garbage Collapsar)
在这个全生命连通周期里不用担心痕迹清理问题：
当你们传完材料，办妥交接，只需要简单直接毫无拘束地——**【集体关掉这页浏览器本身】（或者干脆一点点左侧底部的红色脱离房标）**！
就在全房的最后一片浏览器宣告通讯脱落而人数滑落到 0 的那个微妙刹那：
**埋伏在系统最深处的监管代码回收站机制会被无情引爆。属于你们这个房间密码的全部聊天下文甚至挂靠在云服务深处那层所有原物理附件会被一纸 `rm -rf` 指令连根消散毁灭！不给这台机器乃至接下来的进入者留下半分一毫的追索蛛丝！**
